LAB PRACTICALS

1a. Write a program to accept numbers in an array and calculate the sum of elements at odd and even positions.

1b. Implement Binary Search with a function to find an element’s position or return NULL if not found.

1c. Create a structure for student details and sort the records using the Bubble Sort algorithm.

2. Implement Stack operations (Push, Pop, Display) using arrays and handle overflow and underflow conditions.

3. Implement Queue operations (Insert, Delete, Display) using arrays and handle overflow and underflow conditions.

4. Write a program to create a linked list, store elements entered by the user, display all elements, count the number of nodes, and stop when a NULL pointer is reached.

5. Write a program to insert nodes in a linked list and perform traversal operations.

6. Implement a Binary Search Tree with options to insert, delete, and search elements.

7. Implement DFS traversal on a graph represented by an adjacency list.

8. Implement Kruskal’s Algorithm to find the Minimum Spanning Tree of a weighted graph.

9. Implement Hashing using a seven-slot closed hash table and handle collisions using the hash function (K mod 7).

10. Write a program to copy the contents of one file into another using file handling and generate reports.
